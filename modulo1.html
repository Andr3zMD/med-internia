Módulo 1: Fundamentos de Machine Learning

Del diagnóstico diferencial algorítmico a la predicción de riesgo. Entendiendo cómo las máquinas "aprenden" de los datos clínicos.

Tipos de Aprendizaje en Medicina

Aprendizaje Supervisado

El equivalente al "Caso Clínico Resuelto".

Entrenamos al modelo con datos etiquetados (ej: Radiografías con diagnóstico confirmado de Neumonía). El objetivo es predecir la etiqueta en nuevos pacientes.

Aprendizaje No Supervisado

El equivalente a encontrar "Nuevos Síndromes".

El modelo explora datos sin etiquetas (ej: Analíticas de 10,000 pacientes) para encontrar patrones ocultos o agrupamientos (clusters) desconocidos.

Práctica Clínica: Limpieza de Datos

Antes de aplicar cualquier modelo, debemos curar los datos. En medicina, esto implica manejar valores nulos, estandarizar unidades de laboratorio y anonimizar pacientes.
